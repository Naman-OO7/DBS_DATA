Higher Order Function(HoF)-> The function which take other functions as parameter/ arguments and return other funtions as result or parameter is called higher order function.
{higher order function -> ex. map , reduce ,filter}.
The other function passed as parameter/ arguments is called callback function (CoF).

Map => is an iterative array method executing a transformation callback on every element. It returns a new array of identical length containing modified values without mutating the original source dataset.
Element: The active, individual item from the target array currently undergoing processing within the loop iteration.
Index: The numerical index position representing the current element's precise placement location within the source array.
Array: The original execution context array object upon which the map method was invoked and called.

Filter is an iterative method evaluating every array element against a specified truth test. It returns a brand new shallow copy containing only elements passing the predicate function's true condition.
Element: The specific array element currently being evaluated against the truth criteria of the predicate function.
Index: The active item's sequential index location tracked during the iterative processing of the target array.
Array: The full source array container being structurally traversed and evaluated by the filter iteration process.

Reduce executes a user-supplied reducer callback function on each element of the array sequentially. It completely aggregates these elements into a single accumulated output return value, shifting internal state forward.
Accumulator: The persistent state container storing the cumulative return value from all prior processing execution steps.
Current Value: The individual element within the array currently being systematically processed by the specific reducer logic.
Index: The sequential index integer mapping the current element's specific position during the iterative process traversal.
Array: The target data array object containing all the values traversed by the reduce method structure.
Initial Value: The starting value argument assigned to the accumulator object on the very first loop execution.
